# Project Skills

This directory contains task-specific playbooks used by the agent for high-risk or repeat workflows.

## Available skills
- `new-tenant-scoped-endpoint`: Add/refactor tenant-safe backend endpoints and typed frontend integration.
- `supabase-migration`: Design and apply schema/RLS/index migrations safely.
- `inventory-mutation`: Protect stock integrity across reservation and movement paths.
- `arch-state-update`: Keep `ARCH_STATE.md` synchronized with meaningful architectural changes.
- `order-financial-math`: Enforce `calculateOrderTotals()` as the only totals engine across backend and frontend.
